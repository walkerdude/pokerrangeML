Project Overview

This system uses advanced machine learning to classify opponent hand ranges into four categories: Nuts, Strong, Marginal, and Bluff.

By analyzing betting patterns, position, pot dynamics, and action sequences, the model achieves 99.5% accuracy in predicting hand strength. It's built with 113+ engineered features and multiple ML algorithms including Random Forest, Gradient Boosting, and SVM.

How It Works
1. Data Generation

Simulates realistic poker hands with action sequences, position, pot dynamics, and stack sizes.

2. Feature Engineering

Extracts 113+ meaningful features including aggression ratios, bet sizing patterns, and temporal features.

3. Machine Learning

Trains multiple algorithms (Random Forest, Gradient Boosting, SVM) and selects the best performer.

4. Prediction

Analyzes input data and provides hand strength classification with confidence scores.
